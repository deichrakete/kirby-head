# Kirby  plugin

## Installation

### Download

Download and copy this repository to `/site/plugins/`.

### Git submodule

```
git submodule add https://github.com/foerdeliebe/kirby-head.git site/plugins/head
```

### Composer

```
composer require foerdeliebe/kirby-head
```
